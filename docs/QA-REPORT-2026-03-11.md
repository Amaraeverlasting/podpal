## Feature Detail

### 1. Trial Code Entry — welcome.html
**Status: FIXED (was FAIL)**

**Local check:** `welcome.html` has the trial code section at line 112:
- Input field with placeholder "Have a trial code? Enter it here"
- "Activate Code" button with `onclick="activateCode()"`

**Bug found:** `POST /api/trial/redeem` endpoint did NOT exist in `main.py`. 
The backend only had `GET /trial?code=...` (a redirect flow).
Calling a non-existent endpoint returns 405 Method Not Allowed.

**Fix:** Added `POST /api/trial/redeem` endpoint to `main.py` that:
- Requires auth (returns 401 JSON if not logged in)
- Calls `redeem_trial_code(code, email)`
- Returns `{ success: true, tier, days, expires }` on success
- Returns `{ success: false, error }` on failure

**Verified:** `curl -X POST https://podpal.show/api/trial/redeem -d '{}'` → 400 (missing code) ✅

### 2. Session Viewer
**Status: FIXED (was FAIL)**

**Local check:**
- `session.html` exists in `frontend/dist/` ✅
- `GET /api/session/{session_id}` endpoint at line 750 in `main.py` ✅
- Session.html JS fetches API, handles 401 (redirect to login), 403, 404 ✅
- Dashboard session rows show "View session" link: `<a href="/session?id=${s.session_id}">` ✅

**Bug found:** `HTTPException` was NOT imported in `main.py`.

Import line was:
```python
from fastapi import FastAPI, Request, WebSocket, WebSocketDisconnect
```
The `get_session_detail` endpoint called `raise HTTPException(status_code=401, ...)` causing:
```
NameError: name 'HTTPException' is not defined → 500 Internal Server Error
```

**Fix:** Added `HTTPException` to the fastapi import.

**Verified:**
- `curl https://podpal.show/api/session/test123` → was `500 Internal Server Error`, now `401 {"detail":"Not authenticated"}` ✅
- `/session?id=test123` in browser → loads "PodPal - Session" → JS calls API → 401 → redirects to `/login` ✅

**Note on row clickability:** Dashboard rows have a dedicated "View session" button, not a full-row click handler. Functionally equivalent — the link works. The entire row is not an `<a>` tag, but a button within each row is.
